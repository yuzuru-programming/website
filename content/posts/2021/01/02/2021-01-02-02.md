## ソースコード

{{< code lang="html" title="public/index.html" >}}
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>html</title>
</head>
<body>
  <h1>hello</h1>
</body>
</html>
{{< /code >}}

{{< code lang="json" title="vercel.json" >}}
{
  "routes": [
    { "handle": "filesystem" },
    {
      "src": "/.*",
      "dest": "/index.html"
    }
  ]
}
{{< /code >}}

## コマンド

ローカルで実行 ポート番号3000で鯖が立ち上がる

{{< code lang="bash" title="bash" >}}
$ npx vercel dev
{{< /code >}}

デプロイ
{{< code lang="bash" title="bash" >}}
$ npx vercel --prod
{{< /code >}}
